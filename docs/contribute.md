# Contribute

The SaberCraft Standard is currently in foundation/rewrite mode.

Future contribution paths may include:

- corrections
- notation clarifications
- CM submissions
- glossary improvements
- rules feedback

## Proposed CM submission format

| Field | Required |
|---|---|
| Proposed CM name/ID | Yes |
| Category | Yes |
| Saberist A sequence | Yes |
| Saberist B sequence | Yes |
| Telegraph | Optional |
| Lesson/purpose | Yes |
| Safety notes | Yes |
| Video reference | Optional |
