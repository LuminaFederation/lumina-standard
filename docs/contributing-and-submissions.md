# Contributing and Submissions

## Community growth without losing clarity

LUMINA is designed to support both:

- stable official standards
- community creativity and experimentation

This means schools, instructors, choreographers, and performance groups are encouraged to create original movement libraries using the LUMINA notation system.

At the same time, not every contribution automatically becomes official baseline material.

This page explains the current contribution philosophy.

## What can be contributed?

Examples of possible contributions include:

- original choreography
- notation examples
- diagrams
- teaching notes
- glossary improvements
- clarification proposals
- printable resources
- beginner teaching material
- notation edge cases
- timing conventions
- archival documentation

## Schools can remain independent

Using LUMINA does not require a school to give up:

- branding
- curriculum
- choreography ownership
- teaching style
- internal ranking systems
- performance themes

Schools can publish choreography written in LUMINA notation while still maintaining their own identity.

## Contribution categories

Contributions may eventually fall into different categories.

| Category | Meaning |
|---|---|
| Official | Approved baseline standard material |
| Extended | Official expansions beyond the beginner baseline |
| Community | Independent choreography or contributed material |
| Experimental | Draft systems or unreviewed concepts |

This structure allows experimentation without destabilizing the core notation standard.

## Example

A school might contribute:

- AA-1 Crosswind Exchange
- AA-2 Twin Spiral
- AA-3 Temple Crossing

These sequences could be publicly documented in LUMINA notation while remaining part of that school’s independent choreography library.

The choreography does not become "owned" by the standard simply because it uses the notation language.

## Current contribution philosophy

At this stage of development, the project prioritizes:

- clarity
- consistency
- stable terminology
- readable notation
- beginner accessibility
- long-term preservation

Large changes to the notation system itself should be approached carefully.

## Suggested future submission process

As the standard matures, a future submission process may include:

1. Public proposal submission
2. Notation formatting review
3. Terminology review
4. Safety review
5. Community testing
6. Classification as Official, Extended, Community, or Experimental

The exact process is still evolving.

## Attribution philosophy

Whenever reasonably possible:

- contributors should be credited
- schools should retain attribution
- historical versions should remain documented
- choreography origins should remain visible

One long-term goal of LUMINA is preservation, which includes preserving authorship and historical context.

## Future possibilities

Future community systems may include:

- choreography repositories
- revision history
- public archives
- notation libraries
- printable collections
- cross-school collaboration
- federation partnerships
- community review systems

The current focus is establishing a stable and understandable foundation.

## Key principle

LUMINA should remain open enough to encourage participation, but structured enough to remain readable and durable over time.
