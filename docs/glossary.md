# Glossary

| Term | Definition |
|---|---|
| LUMINA | The choreography standard maintained by Lumina Federation LLC and originated through SaberCraft. |
| Lumen | A practitioner/player within the LUMINA system. |
| Duet | Two Lumens performing together. |
| CM | Choreographed Movement; a reusable choreography block. |
| Telegraph | A silent signal used to call or indicate a CM. |
| Parry | A defensive response to an attack line. |
| LUMINA Core | The official baseline movement library. |
| LUMINA Games | Optional structured game/scoring format. |

!!! note
    Term choices such as **Lumen**, **Lumeneer**, and legacy **Lumener** need to be finalized before v2.0 release.
