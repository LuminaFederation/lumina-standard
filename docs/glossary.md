# Glossary

This glossary defines the working vocabulary of the SaberCraft Standard.

| Term | Definition |
|---|---|
| SaberCraft Standard | The choreography standard maintained by Lumina Federation LLC and originated through SaberCraft. |
| Saberist | The general practitioner term within the SaberCraft Standard — anyone learning or performing SaberCraft Standard choreography. |
| Saberists | Plural of Saberist. |
| Lumen | A player within LUMINA Games specifically. Reserved for the Games layer — not used for general SaberCraft Standard practitioners (see Saberist). |
| Lumens | Plural of Lumen (LUMINA Games players). |
| Duet | Two Saberists performing together. |
| CM | Choreographed Movement; a reusable choreography block. |
| Telegraph | A silent signal used to call or indicate a CM. |
| Attack | A choreographed movement toward an intended target line. |
| Parry | A defensive response to an attack line. |
| Target | A numbered line or point used to describe the intended direction of an attack. |
| SaberCraft Core | The official baseline movement library. |
| LUMINA Games | Optional structured game/scoring format. |

!!! note "Term decision"
    The official general practitioner term is **Saberist**. **Lumen** is reserved specifically for players within LUMINA Games and should not be used for general SaberCraft Standard practitioners. Avoid legacy alternatives such as Lumeneer or Lumener in official standard text unless discussing historical usage.
