# Core Movements

LUMINA Core is the official baseline library of Choreographed Movements, or CMs.

A CM is a reusable choreography block that can be taught, practiced, combined, and documented.

## Categories

| Category | Meaning |
|---|---|
| Core | Official baseline movements |
| Extended | Official additions beyond the beginner core |
| Community | Submitted or adopted community movements |
| Experimental | Under review or not yet official |
