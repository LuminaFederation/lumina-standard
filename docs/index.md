# LUMINA Standard

**LUMINA is a lightsaber choreography standard for recording, teaching, sharing, and safely performing choreographed saber combat.**

Developed and maintained by **Lumina Federation LLC**. Originated through the **SaberCraft** community.

!!! note "Project status"
    This site is being built as the official public home for the LUMINA Standard. Content is currently in foundation/rewrite mode.

## Start here

New to LUMINA? Begin with the [Start Here](start-here.md) guide.

## What LUMINA includes

| Part | Purpose |
|---|---|
| LUMINA Standard | The overall public choreography framework |
| LUMINA Notation | The written movement language |
| LUMINA Core | Official baseline Choreographed Movements |
| LUMINA Games | Optional game/scoring format |
| Glossary | Controlled vocabulary for consistent teaching and documentation |

## Relationship to SaberCraft

SaberCraft is the founding school and original community where LUMINA developed. SaberCraft remains the home for classes, community, events, and training culture.

Read more at [Learn with SaberCraft](learn.md).
