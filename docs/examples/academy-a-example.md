# Example: Academy A Original Choreography

## A fictional example of independent choreography written in SaberCraft notation

This page demonstrates how a school or performance group can use the SaberCraft Standard while maintaining its own choreography library, curriculum, and identity.

The choreography below is fictional.

"Academy A" is an example school created to show how independent groups can document original movement using a shared notation language.

## The important distinction

SaberCraft is the notation language.

The choreography itself belongs to the school using it.

This means schools can:

- create their own movement libraries
- keep their own ranking systems
- teach their own style
- preserve their own choreography

while still using a shared notation structure.

## Example sequence: AA-1 "Crosswind Exchange"

Academy A uses its own naming structure:

- AA-1
- AA-2
- AA-3

instead of SaberCraft’s CM naming structure.

The notation system remains compatible because both schools use SaberCraft.

## Reading the exchange

This sequence demonstrates:

- alternating initiative
- attack-to-parry relationships
- mirrored target responses
- a simple reversal in Step 4

The exchange begins with straightforward attack/parry structure before shifting initiative during the middle of the sequence.

## Why this example matters

The goal of this example is not the choreography itself.

The goal is to demonstrate interoperability.

A SaberCraft instructor could read this sequence.

An Academy A instructor could read a SaberCraft CM.

Both schools remain independent.

The notation language becomes shared.

## A future ecosystem

In a larger choreography ecosystem, schools might eventually publish:

| School | Example library |
|---|---|
| SaberCraft | CM-A through CM-X |
| Academy A | AA-1 through AA-20 |
| Temple Group | Temple Drills |

Each library remains unique.

SaberCraft provides the common structure that makes the choreography readable across communities.

## Key takeaway

The choreography is not the standard.

The notation language is the standard.

That distinction allows schools to collaborate without becoming identical.
